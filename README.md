# VideoPlayer 
一款视频播放 演示Demo
分为两种方式，一种为点击播放：含进度条【缓存进度+播放进度】，播放、暂停、重播
            一种为自动播放，滑动哪一个item  便播放哪一款
            均采用RecyclerView格式的Pager切换
